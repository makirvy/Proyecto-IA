# La Iguana Resort

Sistema administrativo desarrollado utilizando:

- HTML
- CSS
- JavaScript
- GitHub Copilot
- PostgreSQL (estructura preparada)

## Módulos

- Inicio de sesión
- Dashboard
- Visitantes
- Inventario
- Tareas
- Incidencias

## Base de datos

Se preparó una estructura PostgreSQL mediante Node.js y el paquete pg para futuras integraciones.
Actualmente la demostración utiliza localStorage para simplificar la ejecución del proyecto.
